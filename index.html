    $dataX = $("#data-x"),

    $dataWidth.val(data.width);

Getting started

Installation

Include files:

<script src="/path/to/jquery.js"></script><!-- jQuery is required -->
<link  href="/path/to/cropper.css" rel="stylesheet">
<script src="/path/to/cropper.js"></script>

Usage

Initialize with $.fn.cropper method.

HTML:

<!-- Wrap the img with a block element -->
<div>
  <img class="cropper" src="img/picture-1.jpg">
</div>

Javascript:

$(".cropper").cropper({
  aspectRatio: 1.618,
  done: function(data) {
    console.log(data);
  }
});

Notes:

- Please only use the cropper on a visible image. In other words, don't try to use the cropper on a image which was wrapped by a hidden element.

- The size of the cropper was based on the wrapper of the target image, so be sure to wrap the image with a block element.

- The cropper will be re-rendered when the window was resized.

- The result data has be transformed to the true size (based on the natural width and height of the target image).

Options

Setup with $("#target").cropper(options), or global setup with $.fn.cropper.setDefaults(options).

Name Type Default Description
aspectRatio string / number "auto" The aspect ratio of the cropping zone (e.g., "2", "1.3", "0.5", etc.).
Just set it with "auto" to free ratio.
data object {} Options: "x", "y", "width", "height". By default, the cropped zone will appear in the center of the image.
done function function (data) {} The function will be passed an object data and run when the cropping zone was moving.
preview selector undefined A jquery selector, add extra elements for preview.
multiple boolean false By default, this plugin only support one cropper in one page, if you need two or more croppers, you must set this option with true from the second cropper.
modal boolean true Show (true) or hide (false) the black modal layer.
autoCrop boolean true Render the cropping zone automatically when initialize.
dragCrop boolean true Enable to create a new cropping zone when drag on the image.
movable boolean true Enable to move the cropping zone.
resizable boolean true Enable to resize the cropping zone.
minWidth number 0 The minimum width (px of original image) of the cropping zone.
minHeight number 0 The minimum height (px of original image) of the cropping zone.
maxWidth number Infinity The maximum width (px of original image) of the cropping zone.
maxHeight number Infinity The maximum height (px of original image) of the cropping zone.

Methods

reset

  • Reset the cropping zone to the start state.
  • Add a true param to reset the cropping zone to the default state.
  • Usage: $("#target").cropper("reset") or $("#target").cropper("reset", true)

release

  • Release the cropping zone.
  • Usage: $("#target").cropper("release").

destroy

  • Destroy the Cropper and remove the instance form the target image.
  • Usage: $("#target").cropper("destroy").

getData

  • Get the current cropped zone data.
  • Usage: $("#target").cropper("getData").

setData

  • Reset the cropping zone with new data.
  • Param: an object contains "x1", "y1", "width", "height", "x2"(optional), "y2"(optional).
  • Usage: $("#target").cropper("setData", {width: 480, height: 270}).
  • If you want to remove the data, just use like this: $("#target").cropper("setData", {}) or $("#target").cropper("setData", null).

setAspectRatio

  • Enable to reset the aspect ratio after initialized.
  • Param: "auto" or a positive number ("auto" for free ratio).
  • Usage: $("#target").cropper("setAspectRatio", 1.618).

setImgSrc

  • Change the src of the image if need.
  • Usage: $("#target").cropper("setImgSrc", "example.jpg").

getImgInfo

  • Get the image information, contains: "naturalWidth", "naturalHeight", "width", "height", "aspectRatio", "ratio".
  • The "aspectRatio" is the value of "naturalWidth / naturalHeight".
  • The "ratio" is the value of "width / naturalWidth".
  • Usage: $("#target").cropper("getImgInfo").

Events

build.cropper

This event will be fired when the Cropper start to build.

built.cropper

This event will be fired when the Cropper was built.

render.cropper

This event will be fired when the cropping zone was changed by move, resize or crop.

Examples

Cropper in a Bootstrap modal

HTML:

<div class="modal" id="bootstrap-modal">
  ...
  <div class="modal-body">
    <div class="bootstrap-modal-cropper"><img src="img/picture-1.jpg"></div>
  </div>
  ...
</div>

JavaScript:

var $modal = $("#bootstrap-modal"),
    $image = $modal.find(".bootstrap-modal-cropper img"),
    originalData = {};

$modal.on("shown.bs.modal", function() {
  $image.cropper({
    multiple: true,
    data: originalData,
    done: function(data) {
      console.log(data);
    }
  });
}).on("hidden.bs.modal", function() {
  originalData = $image.cropper("getData");
  $image.cropper("destroy");
});
